# BTG Pactual - Plataforma de Gestión de Fondos

## 📋 Descripción
API REST para gestión de fondos de inversión que permite a clientes suscribirse, cancelar suscripciones y ver historial de transacciones.

## 🛠 Tecnologías
- Java 17
- Spring Boot 3.3.5
- MongoDB 1.49.4
- Maven
